# `SIEM key features`

1. Log management
	-	collects and stores vast quantities of log data for analysis

2. event correlation
	-	uses advanced techniques to identify patterns indicating security threats

3. incident monitoring
	-	continuously monitors security events and generates prioritized alerts

4. data retention
	-	retains historical data for compliance and forensic investigations




### scope understanding
-	identify specific security needs of an organization and define how SIEM can address those needs  

		ex:
			-	a health care provider prioritizes patient privacy and regulatory standards
			-	a financial institution prioritizes  detecting fraud and securing transactions

### data correlation rules
-	design and apply rules across all systems and networks


### Asset classification
-	catalog and classify all digital assets for effective monitoring

### BYOD policies
-	establish clear guidlines for personal devices accessing the network
